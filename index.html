故事大綱

內容主要是講述受到號召的英雄們協手對抗突然於世界中央出現的「世界樹」的故事。

每一名英雄看似毫無關係，但他們之間卻隱藏著某種神秘的羈絆；究竟只是命運的捉弄，或者是有誰在背後計畫著一切，

英雄們之間將會迸出怎麼樣的火花，而世界的命運又會是如何?

就讓我們一同在《集結》這款遊戲中，一邊防禦來自「世界樹」的攻擊，一邊試圖解開隱藏那些英雄隱藏在背後的秘密吧!

魔王與勇者之間的遊玩方式差異

玩家在遊玩時，選擇使用魔王或是勇者，遊玩的方式將會不同，

遊玩勇者的玩家可以移動場面上的棋子，與使用不同的技能卡片來對魔王造成傷害，

透過玩家間的互相配合，來盡可能地對魔王造成傷害，以達成遊戲勝利條件

而遊玩魔王的玩家雖然只有一人，但是卻擁有較為強力的攻擊手段，

這裡的每一張卡片都是由我們精心設計與平衡過的，

遊戲中總共有 12 名角色，而每隻角色都有屬於自己的一套卡片，

我們在設計卡片時絕對秉持「技能不重複」、「角色獨特性」、「強度平衡性」這三大遊戲要素來製作，

角色設計與介紹

原定有 12 位角色，但由於集結的遊戲版本一再修改，目前角色與角色立繪皆處於重製狀態，以下展示的角色是目前已經設定完成的角色

角色名稱 : 悠兒

職業 : 舞者

種族 : 半人魚族

年齡 : 人類年齡 22 歲

為了得知自己親生父母的資訊而加入討伐「世界樹」的遠征隊。對周圍的人總是展現出一副冷漠的態度，其實十分喜歡與人聊天，前提是要先有人主動與她搭話

角色名稱 : 法夫納

職業 : 魔法師

種族 : 精靈族

年齡 : 人類年齡 107 歲

沒有人知道法夫納加入遠征隊的原因是什麼，只知道他一開始就在遠征隊之中，由於他強勁的實力，也沒有人敢過問些什麼。雖然他外表是個小孩，但心靈卻早已是個老頭，總是喜歡開一些無趣的老年玩笑，個性也不怎麼討人喜歡，是個有話直說的直腸子角色

角色名稱 : 阿尼瑪

職業 : 上流貴族

種族 : 人魚族

年齡 : 未知

阿尼瑪是「世界樹」遠征隊的號召者，她似乎知道關於「世界樹」的一些隱情，但從來不曾鬆口過任何相關訊息。喜歡人類製作的奢侈品，如包包或是各式金飾等，並不排斥與人類相處，相反反而很喜歡人類。背地裡有許多不利於阿尼瑪的傳聞，有人傳說她對人類友好是為了吸取他們的靈魂、也有傳說她使用禁忌的魔法讓自己永遠維持年輕，沒有人知道哪個傳言是真的，而阿尼瑪本身對這些流言也絲毫不在意

角色名稱 : 未

職業 : 阿尼瑪的護衛

種族 : 人魚族

年齡 : 未知

未與阿尼瑪總是形影不離，她從不主動與其他人對話，唯一肯說話的角色只有阿尼瑪。似乎擁有能夠察覺他人意圖的敏銳直覺，凡是意圖對阿尼瑪不軌或是在私底下散佈對阿尼瑪不好的流言的人，總會被她的觸手施以嚴厲的懲戒
